# Installation

How to installthe cloud deno.

## Clone the repository

First clone the repository, by running

```bash
mkdir deno # or whatever you want to call your directory
cd deno # or whatever you called your directory
curl https://codeload.github.com/iswilljr/cloud/tar.gz/master | \
tar -xz --strip=2 cloud-master/cloud-deno
```

## Install dependencies

<Tabs>
